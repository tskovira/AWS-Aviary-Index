# AWS Aviary Index

A bird information web application built using AWS involving Lambda, API Gateway, MySQL, and S3.

## Overview

This project was developed to demonstrate cloud-based application development using AWS services and modern web technologies. Users can search for bird species, view relevant images, and learn about migration behaviors through a dynamic web interface linked to a database.


## Features

- Search and browse bird species
- View bird images and descriptions
- Explore migration patterns and related information
- Dynamic content retrieval through AWS services
- Responsive user interface
- Serverless cloud architecture

## Technologies

### Cloud Services
- AWS Lambda
- Amazon API Gateway
- Amazon S3
- MySQL Database

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- REST APIs
- Serverless Functions (AWS Lambda)
