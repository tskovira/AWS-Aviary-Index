# AWS Aviary Index

A bird information web application built using AWS using Lambda, API Gateway, MySQL, and S3.

## Features

- Search bird species
- View bird images
- View migration patterns
- Dynamic informative content
- Cloud-hosted architecture

## Technologies

- AWS Lambda
- API Gateway
- MySQL
- S3
- JavaScript
- HTML/CSS
